# Security Policy

## Supported Versions

Only the latest version of Harry-bot is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Harry-bot, please do **not** open a public issue. Instead, please report it privately to the maintainers.

While this is a small project, we take security seriously. Please allow a reasonable amount of time for a response and a fix before disclosing the vulnerability publicly.
